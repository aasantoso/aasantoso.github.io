Phase 0 - The Beginning of All


This is Hacktiv8's first ever Remote Coding Bootcamp program, which is described as "The best Full Stack JavaScript Program in Indonesia" is officialy starting this July.

Specifically, Monday, July 23, 2018.

That being said, I'm actually going to be in the first batch and I'm pretty darn excited!

Anyhow, in this Phase, ergo "Introductory Phase" aka "Fundamental Session", is actually pretty hardcore (already...). So when they said this program is intensive bootcamp, they actually do mean an "INTENSIVE program", no BS guys.

I'm not saying it was ridiculously difficult either. It started with JavaScript 101, pretty fun exercises, then it builds up pretty quickly. LOL.
